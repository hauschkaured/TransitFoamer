# README

Welcome to TransitFoamer! This is a project that was my introduction to making a
program with many features, and is as such imperfect. However, I wish to leave 
it up so that I am able to look back upon it in the future and see how much I
have grown. I hope it can be of some use to the reader.

## Requirements

In the 'data' folder, one must place a document 'apikey.txt' which acts as the 
API key for Pittsburgh Regional Transit's dedicated feed. The static definitions 
are placed in the 'static/' directory. The file is run through the file 'main.py',
and as such can be run via 
```python3 main.py```
